# MyFavoritePlaces_map_app
MyFavoritePlaces is a simple iOS map application built with SwiftUI and MapKit that allows users to see seted favorite locations on the map.

🚀 Features

🗺️ Interactive map built with MapKit

📍 View and explore different locations

🧭 Smooth map navigation

🖼️ Location preview with images and details

📱 Fully responsive UI for different iPhone sizes

⚡ Built using SwiftUI + MVVM architecture

🛠️ Tech Stack

Swift

SwiftUI

MapKit

MVVM Architecture

Xcode

📱 How It Works

The app loads a map using MapKit.

Predefined locations are displayed on the map.

Users can tap a location to see more details.

The map smoothly moves to the selected place.
